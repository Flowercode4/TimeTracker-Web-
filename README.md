# Info
This is a minimalistic extension with support for many browsers that counts your browsing time and stores it in your chrome.storage.local
# Compatibility
- Google Chrome, Opera, Brave, Vivaldi, Yandex Browser
- And other Chromium-based browsers.
# Installation
Download the latest release (.zip) from the [Releases](https://github.com/Flowercode4/TimeTracker-Web-/releases) page

Unpack the archive to a local folder.

Open your browser and navigate to **chrome://extensions/**

Enable Developer mode

Click Load unpacked and select the folder with the extension.
